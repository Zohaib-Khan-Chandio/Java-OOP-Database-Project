## 🛠️ How to Download and Run

Follow these steps to run the project on your local machine using VS Code or any other IDE:

### 1. Download the Project
* Click on the green **"Code"** button at the top right of this GitHub page.
* Select **"Download ZIP"** and extract the files on your computer.
*(Alternatively, you can clone it using: `git clone <repository-url>`)*

### 2. Setup the Database
* Open your MySQL database (like XAMPP/phpMyAdmin or MySQL Workbench).
* Import or copy-paste the queries from the `project.sql` file to create the required tables.

### 3. Run in VS Code / IDE
* Open **VS Code** (or IntelliJ IDEA / Eclipse).
* Install the **"Extension Pack for Java"** if you are using VS Code.
* Open the folder containing `project.java`.
* Update the Database connection credentials (username/password) inside the Java file if needed.
* Click on the **Run** button or press `F5` to execute the program.
